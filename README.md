# CPP_projects
42Vienna C++ projects

There are 10 modules(00-09) that focus on different topics in C++. The projects use C++98 for learning purposes.

## Modules Description
* **CPP00** - Namespace, class, member functions, stdio stream, initialization lists, static, const, and lots of basic stuff
* **CPP01** - Memory allocation, References, Pointers to members, File streams
* **CPP02** - Ad-hoc polymorphism, operators overload and canonical classes
* **CPP03** - Inheritance
* **CPP04** - Subtype polymorphism, abstract classes, interfaces
* **CPP05** - Repetition and Exceptions
* **CPP06** - C++ Casts
* **CPP07** - C++ Templates
* **CPP08** - Templated containers, iterators, algorithms
* **CPP09** - Application of everything learned before, csv manipulations, reverse polish notation calculator and merge-insert sort algorithms
